Project Name: Playgo
Purpose: For sale children's toys
Live URL: https://playgo-3e0bf.web.app

---

Key Features:
1st: First of all, if someone wants to see the details of a popular toy after entering the website, they must sign in. If they are not signed in, they will create an account, that is, they will go to the sign up option.

2nd: If he leaves any input field blank while signing up, he will not be able to proceed to the next step i.e. sign up. Also, if he creates a new account with less than 6 digits and does not have capital letters or small letters, he will be shown an error toast through a toast alert. When the user enters their password during signup, they will be able to see what password they are entering. At the same time, they will be able to see what password they are entering when they sign in.

3rd: When the user is able to sign in, they will be allowed to access the toy product detail page, along with the image from the user's email will show in the navbar on the website, and the user will see their name when hovering over the profile picture.

4th: When the user goes to the toy product details page, they will see a form asking them to submit their name and email. If they do, they will be shown a sweet alert.

5th: If the user forgets their account password, they can go to the Forgot Password section and set a new password.
